# MDP-backward_induction

# Лабораторная работа: Решение МДП методом обратной индукции

**Ссылка на сайт**: `https://andrianovavlada.github.io/MDP-backward_induction/`

## Что делает программа?

Программа принимает файл `data.json` с описанием графа и возвращает:
- Оптимальную политику для каждого состояния
- Значение функции ценности
- Подробные шаги расчета методом обратной индукции
- Выходной текстовый файл

## Формат входного файла

```json
{
  "N": количество состояний,
  "M": количество действий,
  "states": ["наименование состояний",... ],
  "actions": ["наименование действий",...],
  "D": [[доступность действий],...],
  "chance_nodes": [
    {
      "l": узел случайного перехода,
      "from S": текущее состояние,
      "action": действие,
      "to next S": [новые состояния],
      "P": [значение вероятности случайного перехода],
      "R": [значение наград]
    }
  ],
  "gamma": коэффициент дисконтирования,
  "K": горизонт планирования
}
