# MDP-backward_induction

# Лабораторная работа: Решение Марковского Процесса Принятия Решений (МППР) методом обратной индукции для ациклических графов.

**Ссылка на выполнение**: https://andrianovavlada.github.io/MDP-backward_induction/ 

## О программе

Интерактивный калькулятор для решения задач МППР с конечным горизонтом. Реализует алгоритм обратной индукции — динамическое программирование от терминальных состояний к начальному.

Программа принимает файл `data.json` с описанием ациклического графа МППР и возвращает:

- Подробные шаги расчета методом обратной индукции
- Оптимальную политику для каждого состояния
- Значение функции ценности
- Выходной текстовый файл
